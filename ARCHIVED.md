# Archived

Portier CLI is archived and deactivated.

Portier has been taken offline. This repository is retained for historical reference only and must not be released or distributed as an active product unless the Portier service is intentionally restored.

Related shutdown state:

- The Portier backend and relay infrastructure have been deprovisioned.
- Service DNS records for `api.portier.dev`, `api-staging.portier.dev`, and `auth.portier.dev` were removed from the Netlify DNS zone.
- GitHub Actions test, commitlint, and release workflows are disabled at the job level.
